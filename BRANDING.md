# Console Cat - Branding Assets Needed

## Icon Sizes Required
Chrome extensions need specific icon sizes:

### Required Icons
- **16x16** - Extension toolbar icon (small)
- **32x32** - Extension management page
- **48x48** - Extension management page, toolbar icon (medium)
- **128x128** - Chrome Web Store, extension management page

### Design Requirements
- **Theme**: Cat mascot (copycat concept)
- **Style**: Clean, modern, professional
- **Colors**: Should work on both light and dark backgrounds
- **Format**: PNG with transparency
- **Visibility**: Must be recognizable at 16x16 (smallest size)

## Design Concepts

### Option 1: Minimalist Cat Silhouette
- Simple cat head outline
- Console/terminal aesthetic (maybe square brackets or cursor)
- Monochromatic or 2-color design

### Option 2: Cat + Console Elements
- Cat face with console-style elements
- Could incorporate: `>_`, `[]`, or terminal cursor
- Tech-forward look

### Option 3: Abstract/Geometric
- Geometric cat shape
- Modern, flat design
- Bold, recognizable at small sizes

## Color Palette Suggestions
- **Primary**: Blue/teal (tech, trustworthy)
- **Accent**: Orange/amber (cat, warm)
- **Alternative**: Purple/violet (creative, modern)
- **Monochrome**: Grayscale with single accent color
